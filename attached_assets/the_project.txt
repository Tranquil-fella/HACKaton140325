 
Создание анализатора описаний товаров с использованием GigaChat API
Цель проекта
Разработать веб-приложение, которое анализирует качество описаний товаров из CSV-файлов, оценивает их по трем критериям (SEO-оптимизация, полнота информации, читабельность) и предоставляет рекомендации по улучшению.

Технический стек
Язык программирования: PHP
Фронтенд: HTML, CSS (с Bootstrap от Replit - https://cdn.replit.com/agent/bootstrap-agent-dark-theme.min.css), JavaScript
Внешние API: GigaChat API для анализа текста
Формат обрабатываемых данных: CSV-файлы с информацией о товарах (минимум должны содержать поля "Название товара", "Категория" и "Описание")
Основные функции
Загрузка CSV-файлов:

Drag-and-drop интерфейс для загрузки файлов
Валидация типа файла (только CSV)
Отображение информации о загруженном файле
Обработка CSV-данных:

Парсинг данных из CSV
Выделение ключевых полей для анализа
Поддержка для CSV с большим количеством полей (90+)
Интеграция с GigaChat API:

Отправка описаний товаров на анализ
Получение и обработка результатов анализа
Оценка по шкале от 1 до 10 по трем критериям
а так же генерация изображения товара
Представление результатов:

Отображение оценок по каждому критерию
Формирование рекомендаций по улучшению
Возможность скачать результаты в CSV-формате и кртинке в архиве
Детали реализации
Структура файлов
index.php - главная страница с формой загрузки файлов
process.php - обработка загруженного CSV и отправка данных в GigaChat API
results.php - страница с результатами анализа
download.php - скрипт для скачивания результатов
utils/functions.php - утилиты для работы с CSV и API
static/js/main.js - клиентский JavaScript для drag-and-drop и других интерактивных функций
run_php.sh - скрипт для запуска PHP-сервера
Интеграция GigaChat API
Аутентификация с помощью API-ключа (хранится в переменной окружения GIGACHAT_API_KEY)
Запрос к API для анализа описаний по заданным критериям
Обработка ответа и формирование результатов
Интерфейс пользователя
Современный адаптивный дизайн с использованием тёмной темы Replit Bootstrap
Три основных экрана: загрузка файла, процесс анализа, результаты
Информативные сообщения о процессе обработки и результатах
Особенности обработки данных
Корректная обработка кириллических символов в CSV
Поддержка различных форматов CSV (разделители, кодировки)
Оптимизация для обработки больших файлов
Примечания по развертыванию
Для работы требуется доступ к GigaChat API (действующий API-ключ)
Инструкция к API во вложении
